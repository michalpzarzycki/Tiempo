# Tiempo
With the **Tiempo** app, you can look up the weather by city name. 

> **Note:** My [**OpenWeatherMap**](https://openweathermap.org/api) API key is hidden in .env file. If you wanna run my app on your computer, you have to sign up and get your own API key.

## Features
- Search weather by city name
> Gdansk is a perfect city, maybe it'll be your first search ;)
- You can change the temperature units
> if you want to switch between Fahrenheit and Celsius, click a toggle button on the right corner.
